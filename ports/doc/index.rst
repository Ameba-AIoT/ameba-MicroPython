Ameba port documentation
=========================

MicroPython documentation for the Ameba RTOS-M port. Browse the pages below
directly on GitHub, or build them locally with Sphinx (``sphinx-build -b html
ports/doc ports/doc/_build/html``).

Contents
--------

- `General information <general.rst>`_ — port overview, pin naming, Flash and OTA module.
- `Quick reference <quickref.rst>`_ — peripherals, networking and sleep modes with examples.
- `Getting started <tutorial/intro.rst>`_ — installing MicroPython and first steps.
- `Factory reset <tutorial/reset.rst>`_ — restoring the device to a clean state.
